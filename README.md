# gMail clone in React with RSBuild compiler

[original wishlist here](https://github.com/Beej126/GmailZero)... geez, has it been 4 years!?! kids just in high school right now, they'z all that matters =)

- generated with vscode copilot in Agent mode with Claude 3.7 Sonnet model
- prompt: `create a react project named "gMaelstrom" that is a gmail clone, do not nest an extra folder inside of initially chosen workspace folder, typescript build with rsbuild, context data store (no redux), css grid for main page layout, google authentication, and populated wtih actual user emails from gmail cloud`
  - yes! we don't need CRA anymore
  - includes typescript syntax checker plugin (see pluginTypeCheck in rsbuild.config.ts)
- it did a fantastic job
- created the main pages, email list & detail, login
- loaded with my actual emails
- lots of amazing code but several things needing some love
- very decent state of functionality after a weekend... settings menu, loading complex emails with crazy mime embeded images, dark mode looking very decent
